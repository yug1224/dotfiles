# チケット情報取得ルール

引数にチケット ID または URL が渡された場合のみ、以下の手順でチケット情報を取得する。引数がなければスキップする。

## チケットシステムの判別と取得方法

| パターン                | 判別条件                                                                | 取得方法                                                 |
| ----------------------- | ----------------------------------------------------------------------- | -------------------------------------------------------- |
| **Notion**              | URL に `notion.so` / `notion.site` を含む、または Notion ページ ID 形式 | Notion MCP（`notion-fetch`）                             |
| **GitHub Issues / PR**  | URL に `github.com` を含む                                              | GitHub MCP または `gh` CLI                               |
| **その他の URL**        | 上記以外の URL                                                          | URL フェッチで内容を取得                                 |
| **ID のみ（URL なし）** | URL 形式でない文字列                                                    | ユーザーに補足情報を確認するか、会話コンテキストから推測 |

## 取得する情報

- チケットのタイトル・目的
- 背景・要件・受け入れ条件（あれば）
- 関連する技術的コンテキスト
